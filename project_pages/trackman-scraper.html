TrackMan Dashboard Scraper and Plotter

Python script that scrapes private TrackMan dashboard for live pitch and batted ball metrics and then plots IVB vs HB for each pitcher

Python API TrackMan Web Scraping

Overview

I developed a Python script that scrapes live metrics from the TrackMan dashboard. The script initially prompts the user for the pitcher's name and offers the user the chance to switch pitchers, save the plot, switch from warmup to live mode, and end the game. When combined with the Pitch Classification Model I created, it color codes the plotted points of each pitch by pitch color. It also offers the user the chance to override the pitch type in case the model inaccurately classified the pitch type.

Problem

When facing a pitcher for the first time, it is hard to recognize the movement of their pitches. By having this opportunitiy to plot the pitches, it is easier to see the grouping of the pitch's shape for each pitch and aid in the efficiency and accuracy of live in-game tagging.

Methodology

  • Access the private TrackMan dashboard link.
  • Inspected the HTML and the unique class identifications to collect the live metrics.
  • Saves metrics and pitcher's name to a CSV that is later read and used to plot IVB and HB for each pitcher.

Results

Increased efficiency

when tagging live games

Future Improvements

The current way that this script scrapes the TrackMan dashboard website requires the use of Selenium, which can be contigent on the current state of Google Chrome. When Chrome updates, it causes problems with the script. Future versions of this program could look into different ways of web scraping for this type of an API.
